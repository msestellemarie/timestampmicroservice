Timestamp Microservice

Instructions

Pass either a unix timestamp or a natural date as a URL parameter to return both the unix timestamp and the natural date. If you do not pass either, it will return null for both.

Example Input

https://timestamp-microservice-mem.herokuapp.com/August 18, 2018

https://timestamp-microservice-mem.herokuapp.com/1534564800

Example Output

{"natural":"August 18, 2018","unix":1534564800}
